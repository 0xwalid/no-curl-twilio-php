Converts method calls into Twiml verbs.

Converts method calls into Twiml verbs.

A basic example:

.. code-block:: php php> print $this->say('hello'); hello

An example with attributes:

.. code-block:: php print $this->say('hello', array('voice' => 'woman')); hello

You could even just pass in an attributes array, omitting the noun:

.. code-block:: php print $this->gather(array('timeout' => '20'));

:param string $verb: The Twiml verb. :param array $args: - (noun string) - (noun string, attributes array) - (attributes array)

:return: A SimpleXmlElement :rtype: SimpleXmlElement
